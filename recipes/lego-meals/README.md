# Lego Meals

Colección de componentes modulares de comida, altos en proteína y diseñados para congelar en porciones individuales.
Cada receta se prepara una vez, se divide en cubos o porciones, y luego se ensambla con otros componentes según el momento del día.

## Principios generales

- **High protein:** cada comida principal apunta a 20–30 g de proteína por porción; snacks a 8–15 g.
- **Congelación amigable:** se privilegian salsas, guisos, carnes con colágeno, huevos cocidos y masas con baja humedad libre.
- **Recalentado rápido:** las instrucciones indican el método óptimo (microondas, air fryer, horno o sartén).
- **Ensamblado libre:** cada componente puede combinarse con un almidón, una verdura y una salsa para formar un plato completo.

## Equipos recomendados

- Moldes de silicona de ½ o 1 taza (tipo Souper Cubes).
- Air fryer Ninja AF141 o horno de convección para recuperar textura.
- Instant Pot Pro para estofados, carnes y legumbres.
- Papel encerado y bolsas resellables para freezer.

## Combinaciones sugeridas

- **Desayuno rápido:** egg bites + hash de batata + salsa verde.
- **Taco bowl:** tinga de pollo + arroz integral + frijoles negros + aguacate.
- **Plato de cena:** albóndigas griegas + coliflor gratinada + tzatziki fresco.
- **Snack pre-entreno:** energy ball + café o batido de proteína.

## Notas técnicas

- Deja enfriar completamente antes de congelar para evitar condensación y quemadura del freezer.
- Extrae el aire de las bolsas o usa tapa hermética en los moldes.
- Etiqueta cada porción con nombre y fecha; máximo 3 meses en congelador para textura óptima.
- Descongela en refrigerador durante la noche cuando sea posible; evita descongelar a temperatura ambiente.
