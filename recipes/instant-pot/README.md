# Recetas para Instant Pot

Esta carpeta agrupa platillos que explotan la ventaja única de la olla a presión eléctrica: **convertir colágeno en gelatina, extraer sabores acuosos y cocinar almidones/legumbres en una fracción del tiempo**, sin depender de funciones inadecuadas como el modo Sauté para dorar.

## Principios generales

- **No uses Sauté para dorar carne.** El modo Sauté de una Instant Pot no genera suficiente calor seco ni superficie para una reacción de Maillard decente. Si una receta necesita dorado, hazlo en una sartén de hierro fundido o en el horno.
- **No reduzcas dentro de la olla.** La tapa hermética impide la evaporación; para concentrar una salsa, vierte el líquido a una cacerola convencional.
- **Aprovecha la presión alta** (~11.6–12.1 PSI, ~115–120 °C) para:
  - Caldos, fondos y consommés.
  - Cortes con colágeno (costillas, espaldilla, rabo, manitas).
  - Legumbres secas sin remojo.
  - Arroces mantecosos (risottos) y sopas cremosas.
